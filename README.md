## ğŸš€ Cara Menjalankan

### Metode 1: Langsung di Browser

1. Buka folder `UAS`
2. Double-click file `index.html`
3. Website akan terbuka di browser default

### Metode 2: Live Server (Recommended)

1. Install extension **"Live Server"** di VS Code
2. Klik kanan pada `index.html`
3. Pilih **"Open with Live Server"**
4. Website akan terbuka dengan auto-reload saat ada perubahan

### Metode 3: Local Web Server

```bash
# Gunakan Python (jika terinstall)
python -m http.server 8000

# Atau gunakan Node.js http-server
npx http-server
```

Kemudian akses di browser: `http://localhost:8000`

## ğŸ› ï¸ Customization Guide

### Mengubah Color Scheme

Edit file `css/style.css` di bagian CSS Variables:

```css
:root {
  --bs-black: #282a35; /* Background gelap */
  --bs-pink: #ffc0c7; /* Aksen pink */
  --bs-yellow: #fff4a3; /* Aksen kuning */
  --bs-white: #ffffff; /* Background terang */
  --bs-blue: #04aa6d; /* Aksen hijau/biru */
}
```

### Menambah Halaman Materi Baru

1. Buat file HTML baru di folder `materi/BAB[X]/`
2. Copy struktur dari file materi yang sudah ada
3. Update header comment dengan info file
4. Tambahkan link di dropdown menu (semua file HTML)
5. Update prev/next navigation

### Mengubah Font

Ganti di bagian `<head>` semua file HTML:

```html
<link
  href="https://fonts.googleapis.com/css2?family=Inter:wght@400;600;700&display=swap"
  rel="stylesheet"
/>
```

Update di CSS:

```css
body {
  font-family: "Inter", sans-serif;
}
```

## ğŸŒ Hosting di GitHub Pages

### Cara Deploy ke GitHub Pages:

1. **Setup Repository**

   ```bash
   git init
   git add .
   git commit -m "Initial commit"
   git branch -M main
   git remote add origin [URL_REPO]
   git push -u origin main
   ```

2. **Aktifkan GitHub Pages**

   - Buka repository di GitHub
   - Masuk ke Settings â†’ Pages
   - Pilih branch `main` dan folder `/root`
   - Klik Save

3. **Akses Website**
   - Website akan tersedia di: `https://[username].github.io/[repo-name]/`
   - Tunggu beberapa menit untuk deployment pertama

**ğŸ“Œ Catatan:** File README.md akan otomatis ditampilkan di halaman repository GitHub, tapi tidak akan muncul di website yang di-host. README hanya untuk dokumentasi repository.
