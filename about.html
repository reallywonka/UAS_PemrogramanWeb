Tujuan Pembuatan Website

  1. Menerapkan konsep HTML5, CSS3, dan Framework CSS yang telah dipelajari selama perkuliahan.
  2. Membuat website yang responsif dan dapat diakses dengan baik di berbagai perangkat (desktop, tablet, mobile).
  3. Mengimplementasikan best practices dalam web development, termasuk semantic HTML, desain yang user-friendly, dan struktur kode yang rapi.
  4. Menyajikan materi pembelajaran web development secara sistematis dan mudah dipahami untuk membantu sesama pembelajar.
