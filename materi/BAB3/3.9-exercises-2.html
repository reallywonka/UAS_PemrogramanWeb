3.9 Exercises 2: Membuat Layout Responsif Sederhana

Studi kasus ini merupakan integrasi dari seluruh pilar CSS3 yang telah dipelajari sebelumnya. Tujuannya adalah membangun kerangka website portal berita atau profil instansi dengan karakteristik modern menggunakan CSS Grid untuk struktur utama, Flexbox untuk komponen navigasi, dan Media Query untuk adaptasi perangkat.

Perencanaan Struktur (Wireframing)

Tahap awal sebelum pengkodean adalah merencanakan area layout yang terdiri dari lima bagian utama:

  • Header: berisi judul situs.
  • Navigation: berisi link menu utama.
  • Main Content: area berita atau konten utama.
  • Sidebar: berisi informasi tambahan atau iklan.
  • Footer: berisi informasi hak cipta.


Implementasi Kode Sumber (HTML & CSS)

Implementasi ini menggabungkan elemen semantic HTML5 dengan teknik tata letak modern.

1. Struktur HTML (index.html)

Menggunakan elemen semantik di dalam sebuah pembungkus besar bertipe grid-container.

<!DOCTYPE html>
<html lang="id">
    <head>
        <meta charset="UTF-8">
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
        <title>Studi Kasus CSS3 - Kelompok 2</title>
        <link rel="stylesheet" href="style.css">
    </head>
    <body>
        <div class="grid-container">
            <header class="main-header">
                <h1>PORTAL TEKNIK INFORMATIKA UNPAM</h1>
            </header>
    
            <nav class="main-nav">
                <ul class="nav-links">
                    <li><a href="#">Beranda</a></li>
                    <li><a href="#">Kurikulum</a></li>
                    <li><a href="#">Mahasiswa</a></li>
                    <li><a href="#">Kontak</a></li>
                </ul>
            </nav>
    
            <main class="content-area">
                <article class="card">
                    <h2>Pentingnya Penguasaan CSS3</h2>
                    <p>CSS3 bukan sekadar pemberi warna, melainkan arsitek visual dari sebuah aplikasi web...</p>
                    <button class="btn-modern">Baca Selengkapnya</button>
                </article>
            </main>
    
            <aside class="sidebar-area">
                <h3>Agenda Terdekat</h3>
                <ul>
                    <li>UTS Pemrograman Web 1</li>
                    <li>Workshop Tailwind CSS</li>
                </ul>
            </aside>
    
            <footer class="main-footer">
                <p>© 2025 Kelompok 2 - Teknik Informatika S1</p>
            </footer>
        </div>
    </body>
</html>

2. Pengaturan Gaya CSS (style.css)

Kode ini akan menerapkan teknik Grid-Area untuk kemudahan pengaturan responsif.

/* Reset Dasar & Box Model (Sub-bab 3.2) */
* {
    margin: 0;
    padding: 0;
    box-sizing: border-box; /* Memastikan padding tidak menambah lebar */
}

body {
    font-family: 'Segoe UI', Tahoma, Geneva, Verdana, sans-serif;
    line-height: 1.6;
    background-color: #f4f4f4;
}

/* Struktur Grid Utama (Sub-bab 3.5) */
.grid-container {
    display: grid;
    gap: 10px;
    
    /* Layout Mobile: 1 Kolom ke bawah */
    grid-template-areas: 
        "header"
        "nav"
        "main"
        "sidebar"
        "footer";
}

/* Styling Komponen Header */
.main-header {
    grid-area: header;
    background-color: #0044cc;
    color: white;
    padding: 2rem;
    text-align: center;
}

/* Navigasi Menggunakan Flexbox (Sub-bab 3.4) */
.main-nav {
    grid-area: nav;
    background-color: #333;
}

.nav-links {
    display: flex;
    justify-content: center;
    list-style: none;
}

.nav-links a {
    display: block;
    color: white;
    padding: 1rem;
    text-decoration: none;
    transition: 0.3s; /* Sub-bab 3.7 */
}

.nav-links a:hover {
    background-color: #555;
}

/* Styling Main Content & Card */
.content-area {
    grid-area: main;
    padding: 20px;
}

.card {
    background: white;
    padding: 2rem;
    border-radius: 8px;
    box-shadow: 0 4px 8px rgba(0,0,0,0.1); /* Sub-bab 3.7 */
}

/* Tombol dengan Gradasi & Animasi (Sub-bab 3.7) */
.btn-modern {
    margin-top: 1rem;
    padding: 0.8rem 1.5rem;
    border: none;
    border-radius: 5px;
    background: linear-gradient(to right, #0044cc, #0088ff);
    color: white;
    cursor: pointer;
    transition: transform 0.3s ease;
}

.btn-modern:hover {
    transform: translateY(-3px);
}

/* Sidebar & Footer */
.sidebar-area {
    grid-area: sidebar;
    background: #e2e2e2;
    padding: 20px;
}

.main-footer {
    grid-area: footer;
    background: #222;
    color: white;
    text-align: center;
    padding: 1rem;
}

/* RESPONSIVE DESIGN (Sub-bab 3.8) */

/* Untuk Tablet & Laptop (Lebih dari 768px) */
@media screen and (min-width: 768px) {
    .grid-container {
        /* Sidebar di kiri (1fr), Main di kanan (3fr) */
        grid-template-columns: 1fr 3fr; 
        
        grid-template-areas: 
            "header header"
            "nav nav"
            "sidebar main"
            "footer footer";
    }
}

/* Untuk Desktop Besar (Lebih dari 1024px) */
@media screen and (min-width: 1024px) {
    .grid-container {
        max-width: 1100px;
        margin: 20px auto;
        border-radius: 10px;
        overflow: hidden;
    }
}


Bedah Teknis Implementasi

Berdasarkan hasil praktik, terdapat lima poin utama analisis teknis mengapa metode ini digunakan:

  • Penerapan Box-Sizing: Penggunaan border-box sangat krusial agar lebar elemen tetap konsisten meskipun ditambahkan padding.
  • Efisiensi Grid-Area: Teknik ini mempermudah perubahan tata letak pada Media Query hanya dengan mengatur ulang deklarasi area tanpa perlu menghitung kolom secara rumit.
  • Kombinasi Grid dan Flexbox: Grid digunakan untuk kerangka besar (makro layout), sedangkan Flexbox digunakan untuk perataan item navigasi yang lebih fleksibel (mikro layout).
  • Optimasi Visual: Penggunaan gradasi dan bayangan memberikan dimensi visual tanpa memerlukan gambar eksternal, sehingga website tetap ringan.
  • Strategi Mobile-First: Gaya dasar dirancang untuk satu kolom agar perangkat dengan bandwidth rendah memproses kode yang paling sederhana terlebih dahulu.
