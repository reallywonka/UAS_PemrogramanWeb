Pendahuluan HTML

HTML (HyperText Markup Language) adalah bahasa markup standar untuk membuat halaman web. HTML mendeskripsikan struktur dari sebuah halaman web menggunakan markup.

Elemen HTML adalah building blocks dari halaman HTML. Dengan HTML, Anda dapat membuat website Anda sendiri. HTML mudah dipelajari - Anda akan menikmatinya!

Struktur Dasar Dokumen HTML

Setiap dokumen HTML memiliki struktur dasar yang sama. Berikut adalah contoh struktur dasar dokumen HTML: 

<!DOCTYPE html>
<html>
<head>
    <title>Judul Halaman</title>
</head>
<body>

    <h1>Heading Pertama Saya</h1>
    <p>Paragraf pertama saya.</p>

</body>
</html>

Penjelasan:

  • <!DOCTYPE html> - Mendefinisikan dokumen ini sebagai HTML5
  • <html> - Elemen root dari halaman HTML
  • <head> - Berisi meta informasi tentang dokumen
  • <title> - Menentukan judul untuk dokumen
  • <body> - Berisi konten halaman yang terlihat
  • <h1> - Mendefinisikan heading besar
  • <p> - Mendefinisikan paragraf

Tags HTML

Tag HTML adalah nama elemen yang dikelilingi oleh tanda kurung sudut. Tag HTML biasanya berpasangan seperti <p> dan </p>.

Tag pertama dalam pasangan adalah start tag, tag kedua adalah end tag. End tag ditulis seperti start tag, tetapi dengan garis miring sebelum nama tag.

Atribut HTML

Atribut menyediakan informasi tambahan tentang elemen HTML. Atribut selalu ditentukan di start tag dan biasanya datang dalam pasangan nama/nilai seperti: name="value". 

<a href="https://www.contoh.com">Ini adalah link</a>
<img src="gambar.jpg" alt="Deskripsi gambar">

Heading HTML

Heading HTML didefinisikan dengan tag <h1> sampai <h6>. <h1> mendefinisikan heading yang paling penting, sedangkan <h6> mendefinisikan heading yang paling tidak penting. 

<h1>Ini adalah heading 1</h1>
<h2>Ini adalah heading 2</h2>
<h3>Ini adalah heading 3</h3>
<h4>Ini adalah heading 4</h4>
<h5>Ini adalah heading 5</h5>
<h6>Ini adalah heading 6</h6>

Penting: Gunakan heading hanya untuk judul. Jangan gunakan heading untuk membuat teks menjadi BESAR atau tebal.

Paragraf HTML

Paragraf HTML didefinisikan dengan tag <p>. Browser secara otomatis menambahkan spasi (margin) sebelum dan sesudah paragraf. 

<p>Ini adalah paragraf.</p>
<p>Ini adalah paragraf lainnya.</p>

Exercises

Latihan 1: Buat Halaman HTML Sederhana

Cobalah membuat halaman HTML sederhana dengan struktur dasar yang benar:

  1. Buat file baru dengan nama latihan.html
  2. Tulis struktur dasar HTML dengan DOCTYPE
  3. Tambahkan judul "Latihan HTML Saya" di tag title
  4. Di dalam body, buat satu heading h1 dan dua paragraf
  5. Buka file di browser untuk melihat hasilnya

Latihan 2: Gunakan Heading yang Berbeda

Buat dokumen HTML yang menggunakan semua level heading (h1 sampai h6) dengan konten Anda sendiri.
