6.2 Mini Project 2: Membangun Website Company Proofile dengan Bootstrap5

Proyek ini mensimulasikan kebutuhan industri yang mengutamakan kecepatan pengembangan menggunakan Framework Bootstrap 5.

  1. Persiapan dan Struktur Grid

    2. Berbeda dengan CSS murni, Bootstrap menyediakan sistem grid 12 kolom yang memudahkan pengaturan tata letak secara otomatis. Proyek ini membangun profil perusahaan fiksi "TechSolution Nusantara" yang mencakup Hero Section, Layanan, dan Footer.

  2. Implementasi Kode Lengkap

    3. Silakan salin kode di bawah ini dan simpan sebagai file HTML baru (misalnya: proyek_bootstrap.html). Pastikan komputer terkoneksi internet karena kita menggunakan CDN untuk Bootstrap.

  3. Analisis Teknis Kode Bootstrap

    4. Kode di atas menerapkan konsep efisiensi dengan class utility bawaan Bootstrap:

    • Navbar (navbar-expand-lg): Menentukan titik breakpoints di mana menu navigasi akan berubah otomatis menjadi tombol hamburger (pada layar di bawah 992px/Large).
    • Grid System (row & col-md-4): Digunakan pada bagian Layanan untuk menciptakan tiga kotak kartu yang sejajar secara otomatis pada layar medium (tablet/laptop) ke atas. Jika dibuka di HP, kotak akan otomatis turun ke bawah (stacking).
    • Utilities (py-5, shadow-sm): Memberikan padding vertikal (atas-bawah) sebesar level 5 dan efek bayangan tipis secara instan tanpa perlu menulis CSS kustom yang panjang.
    • Icons: Menggunakan Bootstrap Icons yang berbasis vektor dan ringan untuk mempercepat waktu pemuatan halaman dibandingkan menggunakan gambar png/jpg.
