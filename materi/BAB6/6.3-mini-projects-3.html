6.3 Mini Project 3: Membuat Blog Sederhana dengan CSS

Proyek ini bertujuan melatih mahasiswa menggunakan kelas-kelas utilitas Tailwind CSS untuk membangun tata letak Card yang kompleks tanpa menulis file .css tambahan.

  1. Konfigurasi CDN Tailwind

    2. Karena ini adalah proyek skala kecil, kita akan menggunakan CDN script. Namun, perlu dicatat bahwa untuk tahap produksi (profesional), disarankan menggunakan instalasi via NPM/Node.js.

  2. Kode Implementasi Blog

    3. Buat file baru bernama blog_tailwind.html dan salin kode berikut ini:

Perbandingan Pendekatan: Bootstrap vs Tailwind

  • Efisiensi: Bootstrap lebih cepat untuk desain standar dengan komponen jadi (seperti Navbar), sementara Tailwind memberikan kebebasan desain yang lebih unik (utility-first).
  • Responsivitas: Tailwind menggunakan prefix eksplisit seperti md:flex-row untuk mengubah layout secara langsung di dalam kelas HTML, memberikan kontrol yang sangat granular.
