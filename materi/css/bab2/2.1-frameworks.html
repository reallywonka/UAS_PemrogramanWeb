2.1 Pengenalan Bootstrap

Setelah memahami dasar-dasar CSS, sekarang saatnya kita berkenalan dengan CSS Framework. Framework membantu kita membuat desain website yang konsisten dan responsif dengan lebih cepat, tanpa harus menulis semua CSS dari nol.

Apa itu Bootstrap?

Bootstrap adalah framework HTML, CSS, dan JavaScript yang paling populer untuk mengembangkan website yang responsif dan mobile-first.

Kenapa Bootstrap?

Bootstrap menyediakan "komponen siap pakai" seperti tombol, navbar, form, dan kartu (card) yang bisa langsung kamu gunakan hanya dengan menambahkan nama class saja.

Cara Menggunakan Bootstrap

Cara termudah untuk memulai adalah dengan menggunakan CDN (Content Delivery Network). Cukup tambahkan link ini di dalam tag <head> HTML kamu: 

<!-- Bootstrap CSS -->
<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/css/bootstrap.min.css" rel="stylesheet">

Contoh Perbandingan

Membuat tombol warna biru:

Tanpa Bootstrap (CSS Murni):

.tombol-biru {
    background-color: #0d6efd;
    color: white;
    padding: 8px 16px;
    border: none;
    border-radius: 4px;
    cursor: pointer;
}

Dengan Bootstrap:

<button class="btn btn-primary">Klik Saya</button>

Jauh lebih singkat, bukan? Kamu tidak perlu menulis CSS sama sekali untuk mendapatkan tombol yang bagus!

Penting!

Meskipun framework memudahkan, kamu tetap harus mengerti Basic CSS (Box Model, Margin, Padding) agar bisa memodifikasi tampilan Bootstrap sesuai keinginanmu.
