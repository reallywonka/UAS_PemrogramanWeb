Pengenalan JavaScript

JavaScript adalah bahasa pemrograman web. JavaScript dapat mengubah konten HTML, mengubah nilai atribut HTML, mengubah style CSS, dan dapat menyembunyikan atau menampilkan elemen HTML.

JavaScript adalah bahasa pemrograman yang paling populer di dunia. JavaScript mudah dipelajari dan JavaScript bekerja di semua platform.

JavaScript Syntax

JavaScript syntax adalah kumpulan aturan bagaimana program JavaScript dikonstruksi. 

// Ini adalah komentar single-line

/* Ini adalah komentar
   multi-line */

// Statement JavaScript
var x = 5;
var y = 6;
var z = x + y;

JavaScript Statements

Program JavaScript adalah daftar dari "statements" yang akan dieksekusi oleh komputer. 

var x, y, z;    // Statement 1
x = 5;          // Statement 2
y = 6;          // Statement 3
z = x + y;      // Statement 4

JavaScript Variables

Variables adalah "kontainer" untuk menyimpan nilai data. Di JavaScript, Anda dapat mendeklarasikan variables menggunakan var, let, atau const.

var

var x = 5;
var y = "Hello";

let

Diperkenalkan di ES6 (2015), direkomendasikan untuk variable yang nilainya akan berubah:

let x = 5;
let y = "Hello";

const

Digunakan untuk variable yang nilainya tidak akan berubah:

const PI = 3.14159;
const greeting = "Hello World";

JavaScript Functions

Function JavaScript adalah blok kode yang dirancang untuk melakukan tugas tertentu. Function JavaScript dieksekusi ketika "sesuatu" memanggilnya (memanggil/invoke).

Function Syntax

function myFunction(p1, p2) {
    return p1 * p2;   // Function mengembalikan hasil dari p1 * p2
}

// Memanggil function
var result = myFunction(4, 3);  // result = 12

Arrow Functions (ES6)

Arrow function adalah cara singkat untuk menulis function:

// Function tradisional
function hello() {
    return "Hello World!";
}

// Arrow function
const hello = () => {
    return "Hello World!";
}

// Arrow function lebih singkat
const hello = () => "Hello World!";

JavaScript Events

HTML events adalah "hal-hal" yang terjadi pada elemen HTML. Ketika JavaScript digunakan di halaman HTML, JavaScript dapat "bereaksi" terhadap events tersebut.

Event Examples

  • Ketika user mengklik mouse
  • Ketika halaman web telah dimuat
  • Ketika gambar telah dimuat
  • Ketika mouse bergerak di atas elemen
  • Ketika input field diubah
  • Ketika form HTML di-submit

Contoh Event Handler

<button onclick="myFunction()">Click me</button>

<script>
function myFunction() {
    alert("Button was clicked!");
}
</script>

JavaScript DOM

DOM (Document Object Model) adalah programming interface untuk HTML dan XML documents. DOM merepresentasikan halaman sehingga program dapat mengubah struktur, style, dan konten dokumen.

Finding HTML Elements

// Menemukan elemen berdasarkan ID
var element = document.getElementById("myId");

// Menemukan elemen berdasarkan class
var elements = document.getElementsByClassName("myClass");

// Menemukan elemen berdasarkan tag name
var elements = document.getElementsByTagName("p");

// Menemukan elemen dengan CSS selector
var element = document.querySelector(".myClass");
var elements = document.querySelectorAll("p");

Changing HTML Content

// Mengubah konten HTML
document.getElementById("myId").innerHTML = "New content!";

// Mengubah nilai atribut
document.getElementById("myImage").src = "newimage.jpg";

Changing CSS Style

// Mengubah style CSS
document.getElementById("myId").style.color = "red";
document.getElementById("myId").style.fontSize = "20px";
document.getElementById("myId").style.display = "none";
