4.6 Exercises 3: Membuat Landing Page dengan framework CSS

Bagian ini mempraktikkan konversi nyata dari halaman statis HTML dasar (Bab II) menjadi Landing Page modern yang responsive menggunakan Booststrap 5. Trasformasi utama difokuskan pada tiga komponen: navigasi, table, dan formulir.

  1. Persiapan dan Integrasi Library

    2. Langkah awal adalah menghubungkan dokumen HTML dengan library Bootstrap menggunakan metode CDN (Content Delivery Network). Metode ini memungkinkan semua kelas utilitas digunakan secara instan tanpa perlu instalasi file secara lokal di komputer.

    <link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/css/bootstrap.min.css" rel="stylesheet">

  2. Transformasi Struktur (Before vs After)

    3. Dilakukan perbandingan perubahan dari kode HTML murni menuju penggunaan kelas komponen Bootstrap:

    1. Header dan Navigasi: link teks biasa diubah menjadi komponen Navbar yang memiliki fitur sticky (tetap di atas saat di-scroll) dan tombol hamburger responsif untuk layar kecil.
      2. <nav class="navbar navbar-expand-lg navbar-dark bg-primary sticky-top shadow">
    <div class="container">
        <a class="navbar-brand font-weight-bold" href="#">UNPAM AI SEMINAR</a>
        <button class="navbar-toggler" type="button" data-bs-toggle="collapse" data-bs-target="#navbarNav">
            <span class="navbar-toggler-icon"></span>
        </button>
        <div class="collapse navbar-collapse" id="navbarNav">
            <ul class="navbar-nav ms-auto">
                <li class="nav-item"><a class="nav-link" href="#informasi">Informasi</a></li>
                <li class="nav-item"><a class="nav-link" href="#jadwal">Jadwal</a></li>
                <li class="nav-item">
                    <a class="nav-link btn btn-warning text-dark ms-lg-3" href="#daftar">Daftar Sekarang</a>
                </li>
            </ul>
        </div>
    </div>
</nav>

    2. Table Jadwal (Rundown): table yang sebelumnya kaku dimodifikasi dengan kelas .table-hover untuk interaksi kursor dan dibungkus kelas .table-responsive agar tata letak tetap rapi saat dibuka di perangkat mobile.
      3. <div class="table-responsive shadow-sm rounded">
    <table class="table table-striped table-hover align-middle">
        <thead class="table-dark">
            <tr>
                <th>Waktu</th>
                <th>Sesi Acara</th>
                <th>Pembicara</th>
            </tr>
        </thead>
        <tbody>
            <tr>
                <td>08:00 - 09:00</td>
                <td>Registrasi Mahasiswa</td>
                <td>Panitia</td>
            </tr>
        </tbody>
    </table>
</div>

    3. Formulir Pendaftaran: estetika input ditingkatkan menggunakan kelas .form-control dan .form-select, yang memberikan efek visual fokus modern dan validasi visual yang lebih baik.

      4. Hasil Implementasi:

      Proses transfomasi ini menghasilkan halaman web yang profesional melalui penggabungan berbagai komponen Bootstrap 5, seperti Hero Section dengan gradasi, Card untuk informasi fitur, dan sistem grid untuk penempatan konten.
