Pengenalan CSS

CSS (Cascading Style Sheets) adalah bahasa yang digunakan untuk mendeskripsikan presentasi dari dokumen HTML. CSS mendeskripsikan bagaimana elemen HTML harus ditampilkan di layar, di kertas, atau di media lain.

CSS menghemat banyak pekerjaan. CSS dapat mengontrol layout dari beberapa halaman web sekaligus. External stylesheets disimpan dalam file CSS.

Syntax CSS

Aturan CSS terdiri dari selector dan blok deklarasi: 

selector {
    property: value;
    property: value;
}

Penjelasan:

  • Selector - Menunjuk ke elemen HTML yang ingin Anda style
  • Declaration Block - Berisi satu atau lebih deklarasi yang dipisahkan oleh titik koma
  • Setiap deklarasi mencakup nama property CSS dan value, dipisahkan oleh titik dua

Contoh:

p {
    color: red;
    text-align: center;
}

CSS Selectors

CSS selectors digunakan untuk "menemukan" (atau memilih) elemen HTML yang ingin Anda style.

Element Selector

Element selector memilih elemen HTML berdasarkan nama elemen:

p {
    text-align: center;
    color: blue;
}

ID Selector

ID selector menggunakan atribut id dari elemen HTML untuk memilih elemen tertentu. ID dari elemen adalah unik dalam sebuah halaman. Untuk memilih elemen dengan ID tertentu, gunakan karakter hash (#) diikuti oleh id elemen: 

#myid {
    text-align: center;
    color: red;
}

Class Selector

Class selector memilih elemen HTML dengan atribut class tertentu. Untuk memilih elemen dengan class tertentu, gunakan karakter titik (.) diikuti oleh nama class: 

.myclass {
    text-align: center;
    color: green;
}

CSS Colors

Warna di CSS dapat ditentukan dengan menggunakan nama warna yang telah ditentukan, atau nilai RGB, HEX, HSL, RGBA, atau HSLA. 

/* Nama warna */
h1 { color: red; }

/* HEX */
h2 { color: #ff0000; }

/* RGB */
h3 { color: rgb(255, 0, 0); }

/* RGBA (dengan transparansi) */
h4 { color: rgba(255, 0, 0, 0.5); }

CSS Background

Property background CSS digunakan untuk mendefinisikan efek background untuk elemen.

Background Color

body {
    background-color: lightblue;
}

Background Image

body {
    background-image: url("paper.jpg");
}

CSS Box Model

Semua elemen HTML dapat dianggap sebagai kotak. Dalam CSS, istilah "box model" digunakan ketika berbicara tentang desain dan layout.

CSS box model pada dasarnya adalah kotak yang membungkus setiap elemen HTML. Ini terdiri dari:

  • Content - Konten dari box, tempat teks dan gambar muncul
  • Padding - Ruang transparan di sekitar konten
  • Border - Border yang mengelilingi padding dan konten
  • Margin - Ruang transparan di luar border
div {
    width: 300px;
    padding: 20px;
    border: 5px solid black;
    margin: 10px;
}

References

CSS Properties yang Sering Digunakan

Property Deskripsi
color Mengatur warna teks

font-size Mengatur ukuran font

padding Mengatur padding di dalam elemen
border Mengatur border elemen

Exercises

Latihan 1: Styling Teks

Buat file HTML dan CSS terpisah, lalu:

  1. Buat heading h1 dengan warna merah dan center alignment
  2. Buat paragraf dengan warna biru dan ukuran font 18px
  3. Hubungkan file CSS ke file HTML menggunakan tag link

Latihan 2: Box Model

Buat sebuah div dengan:

  • Background color: lightblue
  • Padding: 20px
  • Border: 2px solid black
  • Margin: 10px
  • Width: 300px
