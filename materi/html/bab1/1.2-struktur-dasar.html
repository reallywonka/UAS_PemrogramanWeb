1.2 Struktur Dasar Dokumen HTML

Setiap dokumen HTML memiliki struktur dasar yang sama. Berikut adalah contoh struktur dasar dokumen HTML: 

<!DOCTYPE html>
<html>
<head>
    <title>Judul Halaman</title>
</head>
<body>

    <h1>Heading Pertama Saya</h1>
    <p>Paragraf pertama saya.</p>

</body>
</html>

Komponen Utama Dokumen HTML

1. DOCTYPE Declaration

Deklarasi <!DOCTYPE html> mendefinisikan dokumen ini sebagai HTML5. Ini harus menjadi baris pertama dari setiap dokumen HTML.

2. Element <html>

Element <html> adalah element root yang membungkus seluruh konten HTML. Semua element lainnya harus berada di dalam element ini.

3. Element <head>

Element <head> berisi meta informasi tentang dokumen yang tidak ditampilkan di halaman. Ini termasuk:

  • Judul dokumen (<title>)
  • Link ke stylesheet CSS
  • Meta tags untuk SEO
  • Scripts

4. Element <body>

Element <body> berisi semua konten yang terlihat di halaman web, seperti teks, gambar, link, tabel, list, dll.

Contoh Lengkap

<!DOCTYPE html>
<html lang="id">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Website Saya</title>
</head>
<body>
    <h1>Selamat Datang di Website Saya</h1>
    <p>Ini adalah paragraf pertama saya.</p>
</body>
</html>
