1.1 Pendahuluan HTML

HTML (HyperText Markup Language) adalah bahasa markup standar untuk membuat halaman web. HTML mendeskripsikan struktur dari sebuah halaman web menggunakan markup.

Apa itu HTML?

HTML singkatan dari HyperText Markup Language. HTML adalah bahasa markup standar untuk dokumen yang dirancang untuk ditampilkan di web browser.

  • HTML adalah singkatan dari HyperText Markup Language
  • HTML adalah bahasa markup standar untuk membuat halaman Web
  • HTML mendeskripsikan struktur halaman Web
  • HTML terdiri dari serangkaian elemen
  • Elemen HTML memberitahu browser cara menampilkan konten

Sejarah HTML

HTML pertama kali dikembangkan oleh Tim Berners-Lee pada tahun 1991. Sejak saat itu, HTML telah berkembang melalui beberapa versi:

  • HTML 1.0 (1991) - Versi pertama
  • HTML 2.0 (1995) - Standar pertama
  • HTML 4.01 (1999) - Versi yang sangat populer
  • XHTML (2000) - HTML sebagai XML
  • HTML5 (2014) - Versi terbaru dan paling modern

Mengapa Belajar HTML?

HTML adalah dasar dari semua teknologi web. Berikut alasan mengapa Anda harus mempelajarinya:

  1. Fundamental Web Development - HTML adalah pondasi dari semua website
  2. Mudah Dipelajari - HTML memiliki syntax yang sederhana dan mudah dipahami
  3. Gratis - Anda tidak perlu membayar untuk belajar atau menggunakan HTML
  4. Didukung Semua Browser - HTML bekerja di semua web browser
  5. Karir - Skill HTML sangat dibutuhkan di industri teknologi

Tips

HTML bukanlah bahasa pemrograman, melainkan bahasa markup. HTML menggunakan "tags" untuk menandai berbagai jenis konten.
