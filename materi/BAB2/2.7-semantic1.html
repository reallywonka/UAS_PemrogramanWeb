2.7 Pentingnya Semantic HTML5

Semantic berarti ilmu tentang makna. Menggunakan elemen semantik (bukan sekadar <div>) memberikan keuntungan besar:

  • SEO (Search Engine Optimization): Google memprioritaskan konten di dalam tag semantik (seperti <article>) sebagai informasi utama.
  • Aksesibilitas: Membantu pengguna tunanetra melalui Screen Reader untuk menavigasi struktur halaman dengan cepat.
  • Elemen Utama: Termasuk <header>, <nav> (menu utama), <main> (konten unik), <section> (tema tertentu), <article> (konten mandiri), <aside> (sidebar), dan <footer>.

Implementasi Kode Lengkap:

Berikut adalah contoh struktur website portal berita mahasiswa menggunakan elemen semantik:

<!DOCTYPE html>
<html lang="id">
    <head>
        <meta charset="UTF-8">
        <title>Portal Berita Mahasiswa</title>
    </head>
    <body>
        <!-- HEADER: Bagian kepala website (Logo/Judul) -->
        <header>
            <h1>PORTAL TUGAS KELOMPOK 2</h1>
            <p>Daftar Seluruh Latihan Pemograman Web 1</p>
        </header>
    
        <!-- NAV: Navigasi Menu Utama -->
        <nav>
            <a href="latihan_struktur.html">1. Struktur</a> |
            <a href="latihan_teks.html">2. Teks</a> |
            <a href="latihan_konten.html">3. Link & Gambar</a> |
            <a href="latihan_media.html">4. Tabel & Media</a> |
            <a href="latihan_form.html">5. Formulir</a>
        </nav>
    
        <div class="container">
            <!-- MAIN: Konten Utama Halaman -->
            <main>
                <!-- ARTICLE: Konten mandiri (Artikel berita/blog) -->
                <article>
                    <h2>Selamat Datang di Halaman Utama</h2>
                    <p>Halaman ini menggunakan konsep <strong>Semantic HTML</strong>. 
                    Kamu bisa mencoba klik menu di atas untuk membuka hasil latihan-latihan 
                    sebelumnya tanpa harus membuka folder secara manual.</p>
                    
                    <p>Fitur ini menggunakan <strong>Relative URL</strong>, 
                    yaitu teknik memanggil file yang berada di dalam folder yang sama.</p>
                </article>
            </main>
    
            <!-- ASIDE: Konten Samping / Sidebar (Info tambahan) -->
            <aside>
                <h3>Status File</h3>
                <ul>
                    <li>latihan_struktur.html <span style="color:green">✔</span></li>
                    <li>latihan_teks.html <span style="color:green">✔</span></li>
                    <li>latihan_media.html <span style="color:green">✔</span></li>
                    <li>latihan_form.html <span style="color:green">✔</span></li>
                </ul>
            </aside>
        </div>
    
        <!-- FOOTER: Bagian Kaki Website (Copyright) -->
        <footer>
            <p>© 2025 Kelompok 2 - Universitas Pamulang</p>
        </footer>
    </body>
</html>
