2.8 Exercises 1: Membangun Landing Page Statis

Setelah mempelajari berbagai elemen HTML5 secara terpisah (Teks, List, Link, Gambar, Tabel, Form, dan Multimedia), langkah terakhir dalam bab ini adalah mengintegrasikan seluruh komponen tersebut menjadi satu halaman web yang utuh dan fungsional.

Pada praktik ini, kita akan membangun sebuah Landing Page Seminar Nasional. Proyek ini mencakup kebutuhan administrasi yang lengkap dan menerapkan standar Semantic Structure (Header, Nav, Main, Aside, Footer).



a. Persiapan File

Sebelum memulai pengkodean, pastikan file-file aset berikut telah tersedia di dalam folder kerja (Projek_Makalah_UTS):

  • logo_unpam.png (Logo Universitas)
  • Mars_unpam.mp3 (File Audio)
  • vidio_unpam.mp4 (File Video)

b. Langkah-langkah Praktik

  1. Buka Visual Studio Code.
  2. Buat file baru bernama proyek_akhir_bab2.html.
  3. Salin kode program berikut secara lengkap ke dalam file tersebut.


Kode Program Lengkap:

Salin kode di bawah ini dan simpan sebagai file HTML baru:

<!DOCTYPE html>
<html lang="id">
    <head>
        <meta charset="UTF-8">
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
        <title>Seminar Nasional AI 2025</title>
    </head>
    <body>
        <!-- HEADER: Judul dan Menu Navigasi -->
        <header>
            <img src="logo_unpam.png" alt="Banner Seminar AI" width="10%">
            <h1>SEMINAR NASIONAL: MASA DEPAN ARTIFICIAL INTELLIGENCE</h1>
            <p>
                <!-- Anchor Link ke Bagian Bawah Halaman -->
                <a href="#informasi">Informasi</a> | 
                <a href="#jadwal">Jadwal Acara</a> | 
                <a href="#daftar">Daftar Sekarang</a> |
                <a href="#teaser">Video Teaser</a>
            </p>
            <hr>
        </header>
    
        <!-- MAIN CONTENT -->
        <main>
            <!-- Section 1: Informasi Acara -->
            <section id="informasi">
                <h2>Tentang Acara</h2>
                <p>Seminar ini akan membahas perkembangan teknologi AI terbaru, 
                mulai dari ChatGPT hingga Computer Vision. Peserta akan mendapatkan:</p>
                <ul>
                    <li>Sertifikat Nasional</li>
                    <li>Materi Seminar (PDF)</li>
                    <li>Snack & Makan Siang</li>
                    <li>Networking dengan praktisi IT</li>
                </ul>
            </section>
    
            <hr>
    
            <!-- Section 2: Jadwal (Tabel) -->
            <section id="jadwal">
                <h2>Jadwal Kegiatan (Rundown)</h2>
                <table border="1" cellpadding="8" cellspacing="0" width="100%">
                    <thead>
                    <tr bgcolor="#dddddd">
                        <th>Waktu</th>
                        <th>Kegiatan</th>
                        <th>Pembicara/PJ</th>
                        <th>Lokasi</th>
                    </tr>
                    </thead>
                    <tbody>
                    <tr>
                        <td>08.00 - 09.00</td>
                        <td>Registrasi Ulang</td>
                        <td>Panitia</td>
                        <td rowspan="4" align="center">Auditorium<br>Universitas Pamulang<br>(Viktor)</td>
                    </tr>
                    <tr>
                        <td>09.00 - 10.30</td>
                        <td>Sesi 1: Dasar Machine Learning</td>
                        <td>Dr. Budi Santoso, M.Kom</td>
                    </tr>
                    <tr>
                        <td>10.30 - 12.00</td>
                        <td>Sesi 2: Pemanfaatan AI untuk Bisnis</td>
                        <td>Arya Pranata, S.Kom., M.Kom</td>
                    </tr>
                    <tr>
                        <td>12.00 - 13.00</td>
                        <td colspan="2" align="center"><strong>ISHOMA (Istirahat Sholat Makan)</strong></td>
                    </tr>
                    </tbody>
                </table>
            </section>
    
            <hr>
    
            <!-- Section 3: Form Pendaftaran -->
            <section id="daftar">
                <h2>Formulir Pendaftaran Peserta</h2>
                <p>Silakan isi data diri Anda untuk mengikuti seminar ini:</p>
    
                <form action="proses_daftar.php" method="POST">
                    
                    <label for="nama">Nama Lengkap:</label><br>
                    <input type="text" id="nama" name="nama" placeholder="Masukkan nama lengkap" required size="40"><br><br>
    
                    <label for="email">Email Mahasiswa:</label><br>
                    <input type="email" id="email" name="email" required size="40"><br><br>
    
                    <label>Status Peserta:</label><br>
                    <input type="radio" id="mhs" name="status" value="Mahasiswa" checked>
                    <label for="mhs">Mahasiswa Unpam</label><br>
                    <input type="radio" id="umum" name="status" value="Umum">
                    <label for="umum">Umum</label><br><br>
    
                    <label for="semester">Semester Saat Ini:</label><br>
                    <select id="semester" name="semester">
                    <option value="1">Semester 1</option>
                    <option value="3">Semester 3</option>
                    <option value="5" selected>Semester 5</option>
                    <option value="7">Semester 7</option>
                    <option value="Lulus">Alumni</option>
                    </select><br><br>
    
                    <label>Minat Topik (Boleh pilih lebih dari satu):</label><br>
                    <input type="checkbox" name="minat" value="Web"> Web Development<br>
                    <input type="checkbox" name="minat" value="AI"> Artificial Intelligence<br>
                    <input type="checkbox" name="minat" value="Network"> Jaringan Komputer<br><br>
    
                    <label>Upload Bukti Pembayaran (Gambar):</label><br>
                    <input type="file" name="bukti_bayar"><br><br>
    
                    <button type="submit">Daftar Seminar</button>
                    <button type="reset">Reset Form</button>
                </form>
            </section>
        </main>
    
        <hr>
        
        <!-- ASIDE: Sidebar / Konten Tambahan -->
        <aside id="teaser">
            <h3>Galeri & Dokumentasi</h3>
            
            <p><strong>Teaser Acara (Video):</strong></p>
            <video width="400" controls>
                <source src="vidio_unpam.mp4" type="video/mp4">

            <audio controls>
                <source src="mars_unpam.mp3" type="audio/mpeg">
            </audio>
        </aside>
    
        <hr>

            <center>

            </center>
        </footer>
    </body>
</html>
