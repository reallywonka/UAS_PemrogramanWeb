2.5 Penyajian Data

Pada sub-bab ini, kita akan mempelajari cara menyajikan data yang kompleks dan media interaktif:

  • Struktur Tabel Lanjut: Tabel yang baik harus memenuhi prinsip aksesibilitas dengan pembagian yang jelas:
    - <thead> (Header/Kepala tabel).
    - <tbody> (Body/Data utama).
    - <tfoot> (Footer/Catatan kaki atau total).
    HTML5 juga mendukung penggabungan sel secara vertikal (rowspan) dan horizontal (colspan).

  • Multimedia: Fitur bawaan tag <audio> dan <video> memberikan antarmuka pemutar (Play, Pause, Volume) yang konsisten tanpa bantuan software tambahan (plugin).

Implementasi Kode Lengkap:

Berikut adalah contoh kode untuk membuat Tabel Data Akademik dan menyisipkan Video/Audio:

<!DOCTYPE html>
<html lang="id">
    <head>
        <meta charset="UTF-8">
        <title>Latihan Tabel Lanjut & Multimedia</title>
        <style>
            /* CSS INTERNAL: untuk mempercantik tabel */
            table { 
                width: 80%;
                border-collapse: collapse; 
                margin-bottom: 20px; 
            }
            th, td { 
                border: 1px solid black; 
                padding: 10px; 
                text-align: left;
            }
            th { background-color: #f2f2f2; }
            tfoot { font-weight: bold; background-color: #ddd; }
        </style>
    </head>
    <body>
        <!-- 1. CONTOH TABEL LANJUT -->
        <h2>1. Data Akademik Mahasiswa</h2>
        <table>
            <!-- Kepala Tabel -->
            <thead>
                <tr>
                    <th>No.</th>
                    <th>NIM</th>
                    <th>Nama Lengkap</th>
                    <th>Program Studi</th>
                    <th>Nilai</th>
                </tr>
            </thead>
    
            <!-- Isi Data Tabel -->
            <tbody>
                <tr>
                    <td>1</td>
                    <td>231011400722</td>
                    <td>Egi Witra Aditya</td>
                    <!-- rowspan="3" artinya menggabungkan 3 baris ke bawah -->
                    <td rowspan="3" style="vertical-align: middle;">Teknik Informatika</td>
                    <td>95 (A)</td>
                </tr>
                
                <tr>
                    <td>2</td>
                    <td>231011400741</td>
                    <td>Fajar</td>
                    <!-- Kolom Prodi di sini hilang karena sudah digabung di atas -->
                    <td>95 (A)</td>
                </tr>
                
                <tr>
                    <td>3</td>
                    <td>231011401821</td>
                    <td>Arshanda Amudhia</td>
                    <td>90 (A)</td>
                </tr>
            </tbody>
        </table>
    
        <hr>
    
        <!-- 2. CONTOH MULTIMEDIA -->
        <h2>2. Multimedia Kampus</h2>
    
        <h3>Lagu Mars Universitas Pamulang</h3>
        <!-- Tag Audio dengan kontrol -->
        <audio controls>
            <source src="mars_unpam.mp3" type="audio/mpeg">
            Browser Anda tidak mendukung elemen audio.
        </audio>
    
        <br><br>
    
        <h3>Video Profil Universitas Pamulang</h3>
        <!-- Tag Video dengan kontrol & lebar 500px -->
        <video width="500" controls>
            <source src="vidio_unpam.MP4" type="video/mp4">
            Browser Anda tidak mendukung elemen video.
        </video>
    </body>
</html>
